Twitch JSON API

Built based on the Twitch JSON API challenge from freeCodeCamp
https://www.freecodecamp.com/challenges/use-the-twitchtv-json-api


Details:

Three different categories
Featured - Dota 2 - Coders
App will load the top 10 streams from featured and dota 2 channels to display
Coders category loads individual people as specified by the freeCodeCamp challenge

Goal
- Practice frontend development
- More experience using APIs

Technologies:
HTML, CSS/Bootstrap, JavaScript/jQuery, Twitch API
